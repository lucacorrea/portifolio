# L&J Parallax Canvas Edition — React

Projeto completo em React inspirado no estilo editorial de páginas de lançamento como Shopify Editions, mas com identidade própria para a L&J Soluções Tecnológicas.

## O que vem no projeto

- React + Vite
- Canvas 2D animado no fundo
- GSAP + ScrollTrigger para parallax
- Lenis para rolagem suave
- Imagens remotas da internet via Unsplash
- Componentes separados
- CSS separado e responsivo
- Header fixo com glass effect
- Menu sticky por categorias
- Hero com imagens em camadas
- Galeria/marquee horizontal
- Seções grandes estilo editorial
- Botões de WhatsApp e e-mail

## Como rodar

```bash
npm install
npm run dev
```

Depois abra o endereço mostrado no terminal, normalmente:

```bash
http://localhost:5173
```

## Como gerar versão para hospedagem

```bash
npm run build
```

A pasta final será criada em:

```bash
dist/
```

Envie o conteúdo da pasta `dist` para sua hospedagem.
