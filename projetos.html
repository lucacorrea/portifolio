Sis-Uicam

Descrição:
Sistema de gerenciamento acadêmico para a UIcam, com controle de alunos, professores, cursos e relatórios personalizados.

Tecnologias utilizadas:

  • PHP: Backend para lógica, banco de dados e autenticação.
  • JavaScript: Validações, interatividade e manipulação DOM.
  • HTML5: Estrutura semântica do conteúdo.
  • CSS3: Estilização customizada para o layout.
  • Bootstrap: Grid responsivo e componentes UI prontos.
