# Gestão Comercial - Demo

Sistema de gestão comercial desenvolvido com HTML, CSS e JavaScript.

## Estrutura

```
gestao/
├── index.html
├── README.md
└── assets/
    ├── css/
    │   └── style.css
    └── js/
        └── app.js
```

## Como usar

1. Abra o arquivo `index.html` em seu navegador
2. O sistema está pronto para ser expandido com suas funcionalidades

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
